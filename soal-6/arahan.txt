quqtip arahan:
*install xampp/maamp atau yang lainnya
 disini saya menggunakan xampp
*clone/download folder soal-6 simpan di direktori xampp(C:/xampp/htdocs/)

menggunakan browser localhost/phpmyadmin:
1. buat database pada phpmyadmin(nama database bebas, saran: arkademy_batch_12_5)
2. langsung menuju menu "import" klik button "choose file/pilih file" lalu pilih file
   "arkademy_tb_name_work_salary.sql" pada folder 6_a lalu klik open, jika sudah klik 
   button "Go" dibawah
3. selamat kamu sudah berhasil membuat database dan mengimport tabel
4. sekarang pilih database tadi "arkademy_batch_12_5"
5 lalu ke menu sql dan ketikan query berikut:
  SELECT name.name, work.name as work, salary FROM name INNER JOIN work ON name.id_work = work.id INNER JOIN salary ON name.id_salary = salary.id;
6. sintaks diatas akan mengambil data dari field name di tabel name, field name di tabel work lalu diberi
   alias (as work, agar tampilan field nya tidak sama dengan field name pada tabel name) dan mengambil data
   dari field salary di tabel salary. ketiga tabel tersebut berelasi.

menggunakan command diwindows:
1. buka command pada windows(cmd)
2. arahkan ke directori "C:\xampp\mysql\bin" dengan mengetik "cd C:\xampp\mysql\bin" pada command
3. lalu ketikan "mysql -u root -p" lalu enter. 
4. setelah itu akan diminta untuk memasukan password, masukan password kalian, jika ketika kalian install
   xampp tidak mengeset password maka tekan enter saja(kosongkan).
5. jika sudah maka selamat kamu sudah masuk pada menu database

// cara menbuat database dan import tabel

CREATE DATABASE arkademy_batch_12_5; enter
use arkademy_batch_12_5; enter
buka cmd lagi lalu arah kan ke directori "C:\xampp\mysql\bin" dengan mengetik "cd C:\xampp\mysql\bin" pada
command.
lalu import tabel ke database dengan cara berikut
arahkan ke file .sql "mysql -u root -p < \folder\folder\fileSql.sql"
contoh: 
"mysql -u root -p < D:\arkademyBatch12\batch-5\ArkademyBatch12-5\soal-6\6_a\arkademy_tb_name_work_salary.sql"
lalu enter, masukan password kamu lalu enter(jika tidak set password kosongkan lalu enter).


6. gunakan perintah "show databases;" lalu enter, untuk melihat database apa saja yang pernah kita buat
7. lalu pilih dengan mengetik nama database yang ingin kita gunakan diawali dengan use, 
   contoh: ketik "use arkademy_batch_12_5;" pada command lalu enter, maka kamu akan berada pada database
   tersbut.
8. jika sudah masuk ke database tersebut maka tinggal kita tulis kan query SQL pada command untuk menampilkan
   data yang ada dengan cara custom(hanya menampilkan apa yang ingin kita tampilkan).
9. ketik lah sintaks berikut lalu enter
   SELECT name.name, work.name as work, salary FROM name INNER JOIN work ON name.id_work = work.id INNER JOIN
   salary ON name.id_salary = salary.id;

   maka akan menampilkan data dari relasi/hubungan antara 3 buah tabel.
